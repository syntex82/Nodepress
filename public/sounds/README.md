# Sound Files

This folder contains audio files for the NodePress application.

## Required Files

- `ringtone.mp3` - Played when receiving an incoming video call

## Adding Ringtone

You can add any MP3 file as the ringtone by naming it `ringtone.mp3` and placing it in this folder.

Recommended specifications:
- Format: MP3
- Duration: 3-10 seconds (will loop)
- Sample rate: 44.1kHz
- Bitrate: 128-192kbps

## Free Ringtone Sources

- https://freesound.org/
- https://mixkit.co/free-sound-effects/ringtone/
- https://www.zapsplat.com/
